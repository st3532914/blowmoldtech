# BlowMoldTech - 大型吹塑设备专业平台

BlowMoldTech 是一个专注于大型吹塑设备交易的专业平台，提供设备买卖、租赁、物流等一站式服务。

## 项目结构

```
src/
├── components/        # 公共组件
├── contexts/          # React Context
├── hooks/             # 自定义钩子
├── lib/               # 工具函数
├── pages/             # 页面组件
├── App.tsx            # 应用入口组件
├── index.css          # 全局样式
└── main.tsx           # 程序入口文件
```

## 技术栈

- React 18+
- TypeScript
- Tailwind CSS
- React Router
- Framer Motion (动画效果)
- Recharts (图表)
- Sonner (Toast消息)

## 主要功能

1. **设备市场** - 浏览、搜索和筛选各类吹塑设备
2. **个人中心** - 用户信息管理、订单管理、我的发布
3. **购物车** - 添加、修改、删除购物车商品
4. **订单管理** - 创建订单、支付、物流跟踪
5. **即时聊天** - 与卖家进行沟通
6. **管理员后台** - 设备管理、订单管理、用户管理等
